## *Домашнее задание №10 "Boost.Asio"* 

Программа создает сетевой сервер. От клиентов получает команды в виде строки, формирует блоки команд и результат работы выводит в консоль и записывает в файл.

### Параметры командной строки
```
  bulk_server <port> <bulk_size>
  , где
  port - номер tcp порта для входящих соединений (по умолчанию равен 9000)
  bulk_size - размер статического блока комманд (по умолчанию равен 3)
```
